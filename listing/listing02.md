Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Программа выведет:
2
1

В функции test() устанавливается значение x = 1, затем после return вызывается функция defer, которая увеличивает значение x на 1.
В итоге возвращается значение 2.

В функции anotherTest() устанавливается значение x = 1, затем вызывается return x (Это означает что функция вернет зачение x на данный момент),
далее вызывается функция defer, которая увеличивает значение x на 1, но значение которое должна вернуть функция уже сохранилось.
В итоге возвращается значение 1.

defer <функция> добавляет <функцию> в стек. При возврате функции в которой есть defer, вызываются функции из стека.
Вызов функций из стека производится по правилу "Первый зашел - последний вышел" - значит в обратном порядке.
```
