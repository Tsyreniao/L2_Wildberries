Что выведет программа? Объяснить вывод программы. Объяснить внутреннее устройство интерфейсов и их отличие от пустых интерфейсов.

```go
package main

import (
	"fmt"
	"os"
)

func Foo() error {
	var err *os.PathError = nil
	return err
}

func main() {
	err := Foo()
	fmt.Println(err)
	fmt.Println(err == nil)
}
```

Ответ:
```
Программа выведет:
nil
false

В Foo происходит инициализация переменной err значением nil.
Поле data == nil, поле tab == nil
При возвращении происходит преобразование err к интерфесу error.

В main происходит инициализация переменной err значением nil.
Поле data == nil, поле tab == не пустое

При выводе значения err, будет выведено nil. 
При сравнении err и nil будет false, т.к. interface == nil если тип и значение равны nil (data == nil и tab == nil).

Интерфейсный тип содержит:
tab - метаданные о типе и таблицу методов, иуспользуемых для удоволетворения интерфеса,
data - указатель на область памяти изначального объекта.

Пустой интерфейс - интерфейс, которому удоволетврояет любой тип, т.к. пустой нитерфейсный тип не описиывает методы.
```
